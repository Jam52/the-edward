# The-Edward BnB

A single page web application for BnB business The-Edward.

### Built with

- HTML
- SCSS
- JavaScript
- React

## Installation

### Prerequisites

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/Jam52/the-edward.git
```

2. Install NPM packages

```sh
npm install
```

3. Run tests

```sh
npm test
```

4. Run project

```sh
npm start
```
